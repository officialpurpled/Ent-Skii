🔐 Authentication

Most endpoints require a Bearer token obtained from the login endpoint. Token expires in 20 minutes. 
Authorization: Bearer <your_jwt_token>
Content-Type: application/json

🔑 Authentication Endpoints

Base Path: /api/v1/auth

POST /auth/create-account

Creates a new user account with an automatically generated savings account.

Request Body

FieldTypeRequiredDescription
namestringRequiredFull name
phonestringRequiredPhone number
emailstringRequiredEmail address
bvnnumberRequiredBank Verification Number
ninnumberRequiredNational Identification Number
passwordstringRequiredAccount password
agenumberOptionalUser age (default: 0)
addressstringOptionalUser address

Example Request

{
  "name": "John Doe",
  "phone": "08012345678",
  "email": "johndoe@gmail.com",
  "bvn": 12345678901,
  "nin": 12345678901,
  "password": "securepassword",
  "age": 25,
  "address": "123 Main Street, Lagos"
}

Success Response (201 Created)

{
  "message": "User Registred, Welcome to Entity Bank \n copy your account below",
  "account": 2026250903
}

POST /user/transfer

Fee Structure

Amount RangeFee Calculation
Less than ₦20,000₦10 (base fee)
₦20,000 - ₦49,999₦30 (₦10 + ₦20)
₦50,000 - ₦99,999₦-30 (₦10 - ₦40) ⚠️ Negative fee bug
₦100,000 and above₦100 (₦10 + ₦90)
Classique accountFee waived (₦0)

Request Body

FieldTypeRequiredDescription
amountnumberRequiredAmount to transfer
recievernumberRequiredRecipient's account number
passwordstringRequiredAccount password for verification

Success Response

{
  "message": "transaction succesful",
  "trans_amount": "₦5010 incl. ₦10fee",
  "trxid": "trans-abc123-def456"
}

📝 Changelog

Version 1.0.0 (Current)

✅ Fixed Issues:
  • Phone number validation in /user/data endpoint (now accepts 11-13 digits)
  • Changed /get-accountName to POST /get-receiverName for proper REST convention
⚠️ Known Issues (To Be Fixed):
  • Fee calculation produces negative values for ₦50,000 - ₦99,999 transfers
  • Passwords stored in plain text (should be hashed with bcrypt)
  • No token refresh or logout mechanism
  • Delete account endpoint not implemented
